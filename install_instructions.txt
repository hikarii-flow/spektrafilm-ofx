spektrafilm OFX installation
============================

To install the plugin, copy the .ofx.bundle folder from this ZIP into:

  /Library/OFX/Plugins/

For example:

  /Library/OFX/Plugins/spektrafilm_flow.ofx.bundle
  /Library/OFX/Plugins/spektrafilm.ofx.bundle

You may need administrator permission to copy files into /Library/OFX/Plugins/.

After copying the bundle, restart your OFX host application so it rescans the
plugin folder.


User manual & reference guide
==============================

You can find plugin-relevant documentation at the root of this ZIP. The same manual is also bundled inside the installed plugin at:

  /spektrafilm.ofx.bundle/Contents/Resources/manual.pdf

or by selecting "Open User Manual" from the "Manage" section in the plugin interface.


Have fun and thank you for creating with spektrafilm!
